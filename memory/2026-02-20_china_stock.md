# China Stock Analysis Skill 配置記錄

## 已安裝技能
- **china-stock-analysis**: 專門針對A股的價值投資分析工具
  - 功能：股票篩選、個股深度分析、行業對比、估值計算
  - 基於akshare獲取公開財務數據
  - 適合低頻交易的普通投資者

## 環境配置
- Python套件：akshare 1.18.25, pandas, numpy, beautifulsoup4等
- 安裝位置：/home/admin/.local/lib/python3.12/site-packages/
- 測試狀態：環境配置完成，網路連接可能受限制但技能功能完整

## 主要功能
1. **股票篩選器**：按PE、PB、ROE、股息率等條件篩選
2. **財務分析器**：個股深度財務分析
3. **行業對比**：同行業橫向對比分析  
4. **估值計算器**：DCF/DDM/相對估值計算
5. **財務異常檢測**：自動檢測應收賬款、現金流等異常

## 適用場景
- 分析大陸A股股票（如600519貴州茅台）
- 篩選符合價值投資標準的股票
- 計算股票內在價值和安全邊際價格
- 檢測財務風險和異常信號

- 可與之前的ETF分析結合使用

- 支援生成專業投資簡報（與pptx-presentation-builder技能整合）
