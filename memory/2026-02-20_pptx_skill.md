1. **收集品牌約束** - 顏色、字體、Logo位置
2. **定義簡報結構** - 標題、受眾、目標、時長
3. **生成個別投影片** - 內容、版面、演講備註
4. **審查與優化** - 版面平衡、可讀性、無障礙設計
5. **匯出與交付** - PPTX檔案 + PDF備份
