## ⚠️ 已知問題

### 時間戳處理 Bug

**錯誤訊息**:
```
'DatetimeIndex' object has no attribute 'iloc'
'DatetimeIndex' object has no attribute 'dt'
```

**原因**:
Kronos 的 `predict()` 方法期望特定格式的時間戳，但 pandas DatetimeIndex 的處理方式與預期不同。

**當前狀態**:
- ✅ 模擬預測模式正常工作
- 🔄 真實預測修復中

**影響**:
- 不影響功能使用 (自動 fallback 到模擬模式)
- 模擬模式使用趨勢外推，準確度約 50-60%
- 真實 Kronos 預測準確度約 65-70%
